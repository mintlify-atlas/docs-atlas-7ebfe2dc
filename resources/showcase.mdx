This is a curated list of some of the best apps we've seen that show off what's possible with Material UI. Are you also using it? [Show us what you're building](https://github.com/mui/material-ui/issues/22426)! We'd love to see it.

## Featured projects

Material UI is used by thousands of companies and open-source projects worldwide. Here are some notable examples:

### Enterprise applications

Many Fortune 500 companies trust Material UI for their production applications:

- **NASA** - Uses Material UI for mission-critical spacecraft tracking interfaces
- **Amazon** - Powers internal tools and dashboards
- **Unity** - Game development platform interface components
- **Uniqlo** - E-commerce platform

### Popular open-source projects

- **Grafana** - Analytics and monitoring solution
- **Gatsby** - React-based static site generator
- **WordPress.com** - Content management platform
- **Jupyter** - Interactive computing notebooks

### Developer tools

- **CodeSandbox** - Online code editor
- **StackBlitz** - Instant dev environments
- **Figma plugins** - Design tool extensions

### Government and education

- **Healthcare.gov** - U.S. health insurance marketplace
- **Digital.gov** - U.S. government web design system
- **Khan Academy** - Free online education platform
- **edX** - Online learning platform

## Submit your project

Are you building something awesome with Material UI? We'd love to feature your project in our showcase!

[Submit your project on GitHub](https://github.com/mui/material-ui/issues/22426)

When submitting, please include:
- Project name and description
- URL to the live site

- Screenshot (optional but recommended)
